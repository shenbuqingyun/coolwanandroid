# coolwanandroid
玩Android 首次提交代码 完成首页
所有网络数据来源：鸿洋大神玩Android网站https://www.wanandroid.com/blog/show/2
实现方式：非RxJava实现，用MVC+OKGO实现，主要用于测试网络数据请求展示刷新和一些第三方库的练习。
涉及的第三方库：感谢开源
